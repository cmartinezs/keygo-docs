# RFCs

Actualmente no se mantiene un paquete de RFCs activo en la documentación viva.

## Estado actual

- Las propuestas y paquetes de implementación de 2026-04 fueron archivados para separar decisiones históricas de la documentación operativa actual.
- El material archivado permanece disponible en [../../99-archive/README.md](../../99-archive/README.md).

## Criterio

- Un RFC solo debe vivir aquí mientras esté abierto, vigente y sea necesario para guiar implementación futura.
- Cuando un RFC queda ejecutado, supersedido o solo sirve como contexto histórico, pasa a `99-archive/`.

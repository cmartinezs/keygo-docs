# Archivo

Material histórico, supersedido o útil solo como contexto de decisiones pasadas.

## Qué se archivó

| Ubicación | Contenido |
| --- | --- |
| `legacy-guides/` | Versiones monolíticas antiguas de las guías funcional, técnica y de desarrollo frontend. |
| `2026-04/analysis/` | Auditorías, análisis comparativos y resúmenes de transición. |
| `2026-04/decisions/` | Paquetes antiguos de implementación y RFCs cerrados o históricos. |
| `2026-04/proposals/` | Propuestas no vigentes, gaps históricos y materiales diferidos. |

## Regla

- Nada en esta carpeta es fuente de verdad operativa.
- Si un documento vuelve a ser relevante, debe reintroducirse como documento vivo en su sección correspondiente y no editarse directamente aquí.

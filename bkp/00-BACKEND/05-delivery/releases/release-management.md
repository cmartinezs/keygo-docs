# Release Management

## Fuente de verdad

- Versiones publicadas y cambios: [CHANGELOG.md](../../../CHANGELOG.md)

## Checklist mínimo

1. Verificar build y pruebas relevantes.
2. Confirmar actualizacion de OpenAPI, Postman y guías de integración si hubo cambios de contrato.
3. Confirmar actualizacion de migraciones y modelo de datos si hubo cambios persistentes.
4. Confirmar readiness operativa en `07-operations/`.

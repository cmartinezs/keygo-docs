## Tecnologías y Herramientas
- **Frontend Web:** React, Redux, Tailwind CSS.
- **Backend:** Node.js, Express, MongoDB.
- **Bases de Datos:** MongoDB, PostgreSQL.
- **Mobile:** React Native.
- **OCR:** Microsoft Azure Cognitive Services.
- **Seguridad:** OWASP, JWT, HTTPS.
- **Análisis de Seguridad:** OWASP ZAP, Burp Suite.
- **Control de Versiones:** Git, GitHub.
- **Testing:** Jest, Cypress.
- **Testing de Carga:** JMeter, Locust.
- **Testing Automatizado:** Selenium, Appium.
- **Gestión de Proyectos:** Jira, Confluence.
- **Integración Continua:** GitHub Actions.
- **Despliegue:** Docker, Kubernetes, AWS.
- **Comunicación:** Slack, Zoom.
- **Documentación:** Markdown, Swagger.
- **Monitoreo:** Prometheus, Grafana.
- **Análisis de Código:** SonarQube.
- **Colas de Mensajes:** RabbitMQ, AWS SQS.
- **Almacenamiento** AWS S3
- **Automatización de Infraestructura:** Terraform, Ansible.
- **Documentación de API:** Postman, Swagger.
- **Prototipado y Diseño:** Figma, Adobe XD.
- **Análisis de Logs:** ELK Stack (Elasticsearch, Logstash, Kibana).
- **Gestión de Dependencias:** npm, Yarn.

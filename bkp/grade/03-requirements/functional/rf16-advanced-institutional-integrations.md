# RF16 — Integraciones institucionales avanzadas

**Descripción (alto nivel)**  
GRADE debe ofrecer en fases posteriores integraciones con **sistemas institucionales complejos** para sincronizar información de usuarios, cursos y resultados.

**Alcance incluye (futuro)**
- Integración con **SIS (Student Information System)** para obtener listas de estudiantes.
- Sincronización de calificaciones con sistemas académicos internos.
- Integración con **SSO corporativo** para autenticación unificada.
- Posibles integraciones con plataformas LMS (Moodle, Canvas, Blackboard).

**No incluye (MVP)**
- Adaptadores dedicados inmediatos a cada institución.
- APIs específicas más allá de las generales (RF12).

**Dependencias**
- RF7 (Roles y permisos).
- RF12 (Integración con sistemas externos).
- RNF-INT (Integración segura).

**Criterios de aceptación (futuro)**
1. Es posible importar listas de estudiantes desde un SIS institucional.
2. Las notas generadas en GRADE se sincronizan automáticamente con el sistema académico.
3. Los usuarios acceden con credenciales institucionales vía SSO.
4. Se documenta la interoperabilidad con al menos un LMS común.

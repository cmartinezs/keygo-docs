# RF14 — Preguntas abiertas y calificación con rúbricas

**Descripción (alto nivel)**  
GRADE debe soportar en fases posteriores la **creación y calificación de preguntas abiertas**, ensayos o tareas con criterios de evaluación complejos definidos mediante **rúbricas**.

**Alcance incluye (futuro)**
- Creación de ítems de tipo respuesta abierta.
- Asociación de rúbricas con criterios ponderados para calificación manual o semiautomática.
- Registro de observaciones y retroalimentación cualitativa del docente.
- Generación de reportes que combinen calificación automática y manual.

**No incluye (MVP)**
- Corrección automática mediante IA de texto libre.
- Analítica de calidad de respuestas abiertas.

**Dependencias**
- RF2 (Banco de preguntas).
- RF3 (Gestión de evaluaciones).
- RF6 (Publicación de resultados).

**Criterios de aceptación (futuro)**
1. El docente puede crear una pregunta abierta con rúbrica asociada.
2. El sistema permite ingresar retroalimentación manual y asignar puntaje según la rúbrica.
3. Las notas de ítems abiertos se integran con las de ítems objetivos para calcular el total.
4. Los reportes reflejan tanto resultados numéricos como cualitativos.

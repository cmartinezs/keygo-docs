# Keygo - Unified Documentation

Documentación centralizada y única fuente de verdad para el ecosistema de Keygo (backend y frontend).

Este repositorio será un **submódulo** en los repositorios de backend y frontend para asegurar que todos accedan a la misma información, eliminando redundancia y manteniendo coherencia.

## 📚 Estructura

- **Product**: Contexto del negocio, casos de uso, glossario
- **Functional**: Guías de características, flows, convenciones API
- **Architecture**: Decisiones técnicas, patrones, estrategias
- **Quality**: Testing, code style, observability
- **Operations**: Deployment, runbooks, producción
- **Reference**: Specs, APIs, modelos de datos

## 🎯 Objetivo

Una única fuente de verdad: sin duplicación, sin divergencia, información técnica y producto unificada.

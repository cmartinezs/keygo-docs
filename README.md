# KeyGo Docs

Documentación unificada de Keygo — única fuente de verdad para backend y frontend.

Este repositorio es un **submódulo** de los repos de backend y frontend.

---

## Cómo navegar

La documentación sigue el ciclo de vida del producto (SDLC):

| Carpeta | Fase | Contenido |
|---------|------|-----------|
| `01-product/` | Discovery + Requirements | Problem statement, personas, requisitos RF/RNF |
| `02-design/` | Design & Process + UI | Flujos de sistema, wireframes, design system |
| `03-data-model/` | Data Model | Entidades, ERD, flujos de datos |
| `04-planning/` | Planning | Roadmap, epics, versioning |
| `05-development/` | Development | Arquitectura, APIs, coding standards, ADRs |
| `06-testing/` | Testing | Estrategia de testing, planes, seguridad |
| `07-deployment/` | Deployment | CI/CD, ambientes, release process |
| `08-operations/` | Operations | Runbooks, incident response, SLA |
| `09-monitoring/` | Monitoring | Métricas, alertas, dashboards |
| `10-feedback/` | Feedback | Retrospectivas, user feedback |

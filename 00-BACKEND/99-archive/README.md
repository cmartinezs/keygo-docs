# Archive

Material histórico, reemplazado o conservado solo por referencia.

## Política

- No usar esta sección como fuente de verdad primaria.
- Mantener aqui documentos con valor histórico, propuestas superadas, índices viejos o reportes de trabajo.
- Si un contenido archivado vuelve a ser vigente, debe reintroducirse en la sección canónica correspondiente.

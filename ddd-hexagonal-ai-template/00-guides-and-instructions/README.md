## How to Use This Template

### For Manual Completion
1. Start with discovery (`01-templates/01-discovery/`)
2. Follow the SDLC phases in order
3. Use templates and examples as starting points
4. Complete all COMPLETABLE SECTIONS before moving to the next phase

### For AI-Assisted Completion
1. Review the AI-WORKFLOW-GUIDE.md
2. Use AI agents to generate initial drafts
3. Review and validate generated content
4. Apply project-specific context
5. Sign off on each phase

### Key Rules
- Never skip phases — dependencies flow forward
- Keep historical material in `01-templates/data-input/`
- Maintain phase discipline: discovery/requirements focus on "what", not "how"
- Document assumptions and constraints explicitly

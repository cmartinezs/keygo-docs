# Object Storage

## Description

Binary object storage: S3, GCS, Blob storage.

## When to Use

| Use Case | Recommended |
|---------|-------------|
| User uploads (images, PDFs) | ✅ |
| Static assets | ✅ |
| Backups | ✅ |
| Large files | ✅ |
| Database storage | ❌ Use database |

---

## Characteristics

| Aspect | Object Storage |
|--------|-------------|
| Content | Any binary |
| Size | Up to TB |
| Access | HTTP API |
| Pricing | Per request + storage |

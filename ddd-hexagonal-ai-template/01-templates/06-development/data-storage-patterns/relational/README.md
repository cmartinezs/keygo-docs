# Relational Databases

## Description

SQL databases: PostgreSQL, MySQL, etc.

## When to Use

| Use Case | Recommended |
|---------|-------------|
| ACID transactions | ✅ |
| Complex relationships | ✅ |
| Structured schema | ✅ |
| Reporting/analytics | ✅ |
| Flexible schema | ❌ Consider NoSQL |

---

## Characteristics

| Aspect | Relational |
|--------|-------------|
| Schema | Structured (tables) |
| Transactions | ACID |
| Relationships | Foreign keys |
| Query Language | SQL |
| Scaling | Vertical (mostly) |

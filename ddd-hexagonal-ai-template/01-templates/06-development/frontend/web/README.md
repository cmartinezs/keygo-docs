# Web Applications

## Description

Browser-based applications running in web browsers.

## When to Use

| Use Case | Recommended |
|----------|-------------|
| General purpose apps | ✅ |
| Accessibility required | ✅ |
| Cross-platform web | ✅ |
| Offline-first | ⚠️ Consider PWA |
| Complex graphics | ⚠️ Consider native |

---

## Architecture

| Layer | Responsibility |
|-------|---------------|
| **UI** | Components, screens |
| **State** | Client state, caching |
| **Service** | API communication |
| **Platform** | Browser APIs |

---

## Common Stack

| Layer | Technologies |
|-------|--------------|
| Framework | React, Vue, Angular, Svelte |
| Build | Vite, Webpack, Parcel |
| State | Redux, Zustand, Vuex |
| Data Fetching | TanStack Query, Apollo, SWR |
| Styling | CSS Modules, Tailwind, Styled Components |

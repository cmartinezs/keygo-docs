# Desktop Applications

## Description

Native desktop applications for Windows, macOS, and Linux.

## When to Use

| Use Case | Recommended |
|---------|-------------|
| Native OS integration | ✅ |
| High performance | ✅ |
| Offline operation | ✅ |
| Hardware access | ✅ |
| Web is sufficient | ❌ Use web |

---

## Approaches

| Approach | Description |
|----------|-------------|
| **Native** | Electron, Tauri |
| **Cross-platform** | Flutter Desktop |
| **Web-based** | Desktop wrapper |

---

## Common Stack

| Framework | Description |
|-----------|-------------|
| **Electron** | Node.js + Chromium |
| **Tauri** | Rust + WebView |
| **Flutter** | Dart + native rendering |

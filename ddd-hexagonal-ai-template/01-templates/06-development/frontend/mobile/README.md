# Mobile Applications

## Description

Native or cross-platform mobile applications for iOS and Android.

## When to Use

| Use Case | Recommended |
|---------|-------------|
| Native device features | ✅ |
| Offline operation | ✅ |
| Best UX/performance | ✅ |
| Push notifications | ✅ |
| Simple web app | ❌ Use web |

---

## Approaches

| Approach | Description | Trade-off |
|----------|-------------|-----------|
| **Native** | Swift/Kotlin | Best UX, highest cost |
| **Cross-platform** | React Native, Flutter | Share code, good UX |
| **PWA** | Progressive Web App | Web-based, limited features |

---

## Common Stack

| Approach | Technologies |
|----------|--------------|
| **React Native** | React, Expo |
| **Flutter** | Dart, Flutter |
| **Native** | Swift, Kotlin |
| **PWA** | Service Workers, Web App Manifest |

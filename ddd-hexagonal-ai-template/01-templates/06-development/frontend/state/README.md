# State Management

## Description

Client-side state handling and data flow.

## State Types

| Type | Description | Examples |
|------|-------------|----------|
| **Server State** | Cached server data | User, items |
| **UI State** | Component state | Modal open, selected |
| **Form State** | Form inputs | Form values, errors |
| **Persistent** | Saved locally | Preferences, token |

---

## Patterns

| Pattern | Use Case |
|---------|----------|
| **Context** | Simple global state |
| **Store** | Complex global state |
| **Local State** | Component-specific |
| **Server Cache** | API data caching |

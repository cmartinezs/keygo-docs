# Authorization

## Description

Access control patterns: RBAC, permissions, scope validation.

## When to Use

| Use Case | Recommended |
|---------|-------------|
| Role-based access | ✅ |
| Resource-level permissions | ✅ |
| Organization isolation | ✅ |
| No access control needed | Skip this |

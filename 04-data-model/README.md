# Data Model

Esta sección documenta el modelo de datos de Keygo: las entidades del dominio, sus atributos y relaciones, y cómo la información fluye a través del sistema durante las operaciones principales.

El modelo de datos se deriva directamente de los bounded contexts definidos en la fase de diseño. Cada entidad corresponde a un concepto del dominio con identidad propia y ciclo de vida definido.

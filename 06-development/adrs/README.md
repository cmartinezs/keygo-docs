## Introducción

Los ADRs documentan decisiones arquitectónicas significativas, incluyendo:
- El contexto y problema
- La decisión tomada
- Las consecuencias (positivas y negativas)
- Estado (aceptado, deprecado, reemplazado)

Formato basado en [Markdown ADR](https://github.com/joelparkerhead/markdown-adr).
